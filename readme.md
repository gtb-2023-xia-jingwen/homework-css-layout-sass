## 技术提示

使用 sass 语法实现样式，此问题需要修改 _public/styles_ 目录下的 index.scss 文件  
_public/styles_ 目录下提供了 \_reboot.scss，\_variables.scss，\_mixins.scss 文件，请根据需要确定是否采用。

字体，颜色，宽高，间距大概一致即可，不必精确到像素级别

## 下载依赖

```bash
$ npm install
```

## 运行网页

请执行如下代码运行网站并打开页面：

```bash
$ npm start
```

之后在浏览器中访问：http://localhost:8080
